# Infraestrutura Labs

Repositório dedicado ao estudo e documentação de laboratórios práticos de infraestrutura de TI, redes, virtualização, Windows Server e computação em nuvem.

## Objetivo

Desenvolver conhecimentos técnicos voltados para administração de ambientes corporativos, seguindo uma trilha de aprendizado focada em:

* Redes TCP/IP
* Windows Server
* Active Directory
* Group Policy (GPO)
* Virtualização
* Microsoft Azure
* Microsoft 365
* Intune e Endpoint Manager
* Segurança de Redes

## Estrutura do Repositório

### Redes

Documentação de conceitos fundamentais de redes, protocolos e serviços.

### Virtualização

Laboratórios utilizando VirtualBox, Hyper-V e outras tecnologias.

### Windows Server

Instalação, configuração e administração de servidores Windows.

### Active Directory

Gerenciamento de usuários, grupos, permissões e políticas.

### Cloud

Laboratórios e estudos em Microsoft Azure e serviços relacionados.
