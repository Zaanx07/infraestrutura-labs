# Redes

Esta pasta reúne estudos, anotações e laboratórios práticos relacionados aos fundamentos de redes de computadores.

## Objetivo

Compreender os conceitos essenciais de redes TCP/IP e desenvolver conhecimentos aplicáveis a ambientes corporativos de infraestrutura de TI.
